Welcome to Laptop Price Prediction

About the Model

Welcome to our Laptop Price Prediction tool! This model was built using a comprehensive dataset that included various attributes of laptops such as the Company, Product Type, Screen Size, Screen Resolution, CPU, RAM, and more.

Data Cleaning and Feature Engineering

Initially, our dataset had the following columns:

  • Laptop ID
  • Company
  • Product
  • Type Name
  • Inches
  • Screen Resolution
  • CPU
  • RAM
  • Memory
  • GPU
  • Operating System
  • Weight
  • Price in Euros

Final Features

After cleaning, we focused on the following key columns:

  • Company: The brand of the laptop.
  • TypeName: The type of the laptop (e.g., Ultrabook, Gaming).
  • Inches: Screen size in inches.
  • Screen Resolution: The resolution of the laptop screen.
  • Operating System: The OS installed on the laptop.
  • CPU Company: Extracted from the CPU name.
  • Processor Generation/Series: Identified from the CPU details.
  • RAM in GB: The amount of RAM in gigabytes.
  • ROM: The type of storage (SSD, HDD).
  • Storage Drive: The storage capacity in bytes.
  • Price in Rupees: Converted from Euros for local relevance.
